# Path of the Lost Mechanics Cheatsheet

## Interactions and Menu
- Use spacebar for interactions such as:
  -  continuing or closing textbox
  -  initiating conversation with an NPC
  -  interacting with certain objects
- Use escape to access the pause/menu screen

## Movement and Abilities
- Use keys WASD to move the player
- Once unlocked hold the left mouse button to draw an arrow and relase the button to fire
- Once unlocked press shift to use the player's dash ability
- Once unlocked mouse over an object and hold right mouse button to pick up and move an object with telekinesis. Release the button to drop the object.

## Powerups
- Hearts
  - Walking into to a heart object will pick it up, heal the player completely, and increase the player's max health by one heart
- Gear
  - Walking into the gloves or the boots will pick them up and enable the abilities

## Known Bugs
- On Mac scaling issue causes cursor to appear too large.
